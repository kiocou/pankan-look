# pankan-look

桌面云媒体中心 · Tauri 2 + React 18 + TypeScript

> 本仓库为「盘盘看」项目的重构版（kiocou/pankan-look）。
> 第一版代码基于 `盘盘看` 原始项目文档逆向 + 重构。

## 快速开始

```bash
cd panpankan
npm install
npm run tauri:dev
```

详见 [README.md](README.md) 和 [SPEC.md](SPEC.md)。

## 目录

- `panpankan/` —— Tauri 主工程
- `panpankan/src-tauri/` —— Rust 后端
- `panpankan/frontend/` —— React 前端
